# Good Practices for Software Development Assignment

1. Which point(s) were new to you?

1. The importance of over-communication in software development.
2. Joining meetings early to build better team relationships.
3. Combining all questions into a single organized message.
4. The idea that programming mainly depends on sustained attention and deep work.
5. Using screenshots, Loom videos, and GitHub Gists while asking technical questions.

2. Which area do you think you need to improve on? What are your ideas to make progress in that area?  
Focus management and communication skills.
1. Reduce social media usage during study and work hours.
2. Practice deep work daily with fixed time blocks.
3. Communicate project updates more clearly and regularly.
4. Ask better technical questions with proper details.
5. Improve note-taking during meetings and discussions.
6. Keep a daily routine for learning and coding practice.
7. Participate more actively in team discussions.
8. Maintain consistency in completing tasks on time.
